# How to Inspect the Database

Audience: Developers.

Prisma Studio provides a GUI for your database.

## Steps

1.  **Start Studio**
    ```bash
    cd web
    npx prisma studio
    ```

2.  **Browse**
    Open `http://localhost:5555`.
    You can view `Entity`, `RawIngestion`, and other tables directly.

Evidence: Standard Prisma tooling.
